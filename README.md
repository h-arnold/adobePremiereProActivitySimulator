# Premiere Pro Workflow Simulator

A small Windows PowerShell starter project for scripting and automation.

## Requirements
- Windows PowerShell 5.1 or PowerShell 7+ on Windows

## Run

```powershell
powershell -ExecutionPolicy Bypass -File .\main.ps1
```

If you are using PowerShell 7:

```powershell
pwsh -File .\main.ps1
```

## Project Files
- `main.ps1` - entry point script
- `.vscode/tasks.json` - VS Code task for running the script
- `.github/copilot-instructions.md` - workspace instructions for Copilot
